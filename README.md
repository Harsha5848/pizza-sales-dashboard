# Pizza Sales Dashboard

## Project Overview
This Power BI dashboard analyses pizza sales data to identify revenue trends, top-selling pizzas, and customer purchasing patterns.

## Tools Used
- Power BI
- Excel
- DAX

## Dashboard Insights
- Total Revenue
- Total Orders
- Top 10 Pizzas
- Sales by Category
- Monthly Sales Trend
